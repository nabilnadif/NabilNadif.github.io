Nadif PORTFOLIO

Saya Muhammad Nabil Nadif, saya merupakan seorang mahasiswa di fakultas teknik informatika Universitas Riau angkatan 2024. Saya sangat menyukai dunia teknologi dan sangat menyukai dunia programming. Saya sangat senang belajar hal baru dan sangat senang berbagi ilmu kepada orang lain. Saat ini saya sedang belajar menjadi seorang web developer dengan membangun sebuah web portofolio pribadi.

Alasan saya memilih teknik informatika sebagai program studi di perkulihan saya adalah karena saya memiliki ketertarikan dengan perkembangan zaman yang selalu melibatkan perkembangan teknologi. Sehingga manusia harus dapat menyertai kehebatan dan perkembangan pesat dari teknologi dari masa ke masa. Dengan berkuliah di teknik informatika ini saya berharap saya dapat menjadi orang yang hebat dalam menggunakan bahkan menjadi perancang dari sebuah kemajuan teknologi di masa mendatang.
